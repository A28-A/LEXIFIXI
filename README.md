# ✍️ LexiFix AI (Lite)
**A Pure Python Spelling & Grammar Corrector with a Pastel Twist.**

LexiFix is a lightweight web application that uses deep learning (T5 Transformers) and natural language processing to fix grammatical errors and spelling mistakes instantly. It features a modern, calming pastel-themed interface designed for a seamless user experience.



---

## ✨ Key Features
* **Dual-Pass Correction:** Uses `pyspellchecker` for rapid typo fixing followed by a T5 Transformer model for advanced grammar correction.
* **Modern Pastel UI:** A soft, clean design using Mint, Lavender, and Peach tones.
* **One-Click Copy:** Quickly copy corrected text to your clipboard.
* **Privacy-Focused:** Designed to run in **Offline Mode** once the model is downloaded.
